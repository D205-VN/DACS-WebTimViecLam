Backend MVC layout:

- `src/models/`: database schemas, persistence models, and domain data.
- `src/controllers/`: Express request handlers.
- `src/services/`: business logic, schedulers, email, blockchain, and AI integrations.
- `src/repositories/`: SQL query modules used by services.
- `src/routes/`: Express route definitions and route registry.
- `src/core/`: shared middleware, config, errors, realtime, and utility code.
- `src/infrastructure/`: database and external infrastructure adapters.
